Selfbot Commands

Useful:

  • +crypto: Sends my crypto address
  • +server: Sends an invite to my discord server
  • +shop: Sends my sellpass shop
  • +pfp: Displays the pfp of a user
  • +banner: Displays the server banner
  • +serverlogo: Displays the server logo
  • +membercount: Show the amount of members in the server
  • +snipe: Snipe the last deleted message
  • +afk/afkoff: Enable/Disable afk mode

Fun:

  • +image: Generates a random image
  • +joke: Tells a joke
  • +8ball: Answers a question with a random answer
  • +coinflip: Flips a coin
  • +dice: Rolls a dice
  • +fact: Tells a random fact about animals
  • +iq: Guesses someone's IQ
  • +qr: Turns your text into a QR Code

Troll:

  • +fakeinvite: Generates a fake invite

Help:

  • +commands: Shows this message
  • +ping: Shows the bot ping
