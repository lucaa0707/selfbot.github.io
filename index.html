Selfbot Commands

Useful:

  • +crypto: Sends my crypto address
  • +server: Sends an invite to my Discord server
  • +shop: Sends my sellpass shop
  • +pfp: Displays the profile picture of a user Usage: +pfp @user
  • +banner: Displays the server banner
  • +serverlogo: Displays the server logo
  • +membercount: Shows the number of members in the server
  • +snipe: Snipes the last deleted message
  • +afk/afkoff: Enable/Disable AFK mode Usage: +afk reason

Fun:

  • +image: Generates a random image
  • +joke: Tells a joke
  • +8ball: Answers a question with a random response Usage: +8ball question
  • +coinflip: Flips a coin
  • +dice: Rolls a dice
  • +fact: Tells a random fact about animals
  • +iq: Guesses someone's IQ Usage: +iq @user
  • +qr: Turns your text into a QR Code Usage +qr your text

Troll:

  • +fakeinvite: Generates a fake invite Usage: +fakeinvite discord https://discord.gg/D3Wcy9vVj9

Help:

  • +commands: Shows this message
  • +ping: Shows the bot ping
